"Loading..."

Web Art

2016


This project extracts loading icons from their usual contexts and creates a new space around them. Using color gradients and computer system sounds, I aimed at constructing endless loops that transform the dullness of the loading image to something visually and sonically stimulating.

The project explores the interplay between minimal visual and audio. Nothing is heavily edited, but different elements were layered upon one another that amplify their simplicities. The viewers are able to create new windows and combine the different loading worlds in their own ways.
